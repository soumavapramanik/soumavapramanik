### 🔬 How I Work

```text
Hardware Signal  →  Data Pipeline  →  Model  →  Audit & Verify  →  Edge Deployment
   (sensors)        (clean + ETL)    (train)   (faithfulness)      (embedded)
```

- **Data first.** Before modeling, I audit. Garbage in, garbage out — so I write scripts that catch the garbage early.
- **Reliability over hype.** A model I can't explain or trust isn't done. I verify faithfulness, not just accuracy.
- **Close to the metal.** My ECS background means I care about what happens after the notebook — latency, memory, and getting models onto real hardware.
